7. Blueprint Reference | Event Instance

These methods are used to control individual Event Instances.

Methods:

  • Event Instance Get Parameter Get a parameter value on an FMOD Event Instance.
  • Event Instance Get Parameter Value Get a parameter value on an FMOD Event Instance.
  • Event Instance Is Valid Return whether this FMOD Event Instance is valid. The instance will be invalidated when the sound stops.
  • Event Instance Play Play an FMOD Event Instance.
  • Event Instance Release Release an FMOD Event Instance.
  • Event Instance Set Parameter Set a parameter on an FMOD Event Instance.
  • Event Instance Set Paused Pause/Unpause an FMOD Event Instance.
  • Event Instance Set Pitch Set pitch on an FMOD Event Instance.
  • Event Instance Set Property Set an FMOD event property on an FMOD Event Instance.
  • Event Instance Set Transform Set 3D attributes on an FMOD Event Instance.
  • Event Instance Set Volume Set volume on an FMOD Event Instance.
  • Event Instance Stop Mute/Unmute the bus.
  • Event Instance Key Off Allow an FMOD Event Instance to continue past a sustain point.

Event Instance Get Parameter

Get a parameter value on an FMOD Event Instance.

C++

static float EventInstanceGetParameter(
  FFMODEventInstance EventInstance,
  FName Name
);
EventInstance
FFMODEventInstance to use.
Name
Name of parameter.

Will be deprecated in FMOD 2.01, use Event Instance Get Parameter Value instead.

Event Instance Get Parameter Value

Get a parameter value on an FMOD Event Instance.

C++

static void EventInstanceGetParameterValue(
  FFMODEventInstance EventInstance,
  FName Name,
  float &UserValue,
  float &FinalValue
);
EventInstance
FFMODEventInstance to use.
Name
Name of parameter.
UserValue
Parameter value as set from the public API.
FinalValue
Final combined parameter value.
